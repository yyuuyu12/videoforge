# 竞品账号风格拆解与差距清单（2026-07-23）

方法：每条抽 2 帧直读（20 帧）+ 3 份文案 + 音频频谱对比（竞品 vs job-30 成片）。

## 一、账号风格画像

**主体形态（按出镜频率）**
1. **AI 生成数字人全屏出镜**为主画面（07/10 号片全屏坐播，两侧浮动 app 图标道具卡）——"人"是画面主体而非角落小窗
2. **黑底发光手绘流程图**（08 号：icon 节点+发光描边+红色失败路径+虚线信号流；03 号：科技感节点卡+3D 透视+呼吸灯）——出现频率最高的图解套路
3. **屏幕录制全屏直放**（01/02 号：无限画布/终端界面），重点区域用**局部 UI 放大浮层**（02 号输入框放大卡骑在屏录上）
4. **素材高密度拼贴**：视频截帧三卡连线图解（01 号）、海报拼贴墙（04 号）、视频画廊网格（05 号）
5. **AI 伪实拍 B-roll / 剧情段**（03/06 号：办公室场景+特效数据流、半透明虚拟角色情景剧）

**字幕系统**：底部居中、白字黑底板（或黑描边）、单行 8-14 字、约占画面高 5-6%；剧情段带中英双语+括号台词。**没有花字/变色**——字幕朴素，重点全靠画面。

**声音**：频谱实证**全程 BGM 底乐**（连续能量带铺满语音间隙）；我们的成片是纯人声，段间静默肉眼可见。

**文案**：算账式/对照实验式开场（"为什么报价不低利润薄""看完你记住了什么"）、公式化拆解、结尾 CTA 单独成卡。

**色彩质感**：炭黑网格底 + 单强调色（绿/红）为主，白色发光描边；整体"暗房里的发光图纸"感。

## 二、我们已对齐的（不用再投）

- 炭黑底+强调色、硬切为主、镜头推拉节奏、字幕契约（他们的字幕并不比我们强）
- MediaFrame 容器、host-full/host 全屏时刻机制、SFX 音效层（刚落地）

## 三、差距与优化方向（按性价比排序）

| # | 差距 | 对应能力 | 做法（全部确定性/组件层） |
|---|---|---|---|
| 1 | **发光流程图**——竞品最高频套路，我们的自绘 SVG 朴素 | 新效果件 `FlowDiagram` | 节点/边声明式数据驱动：icon+发光描边+虚线流动+红色失败路径+逐节点生长（踩口播 word 触发）；进 chapterLint 词表与 EXEMPLARS |
| 2 | **BGM 底乐层** | render 混音 | 与 SFX 同构：BGM 轨循环铺底 + 语音 sidechain ducking（人声时 -12dB）；免版权曲库 3-5 首按主题选；`config.bgm` 开关。建议从 P2 提前到 P1 |
| 3 | **素材组合版式**——拼贴墙/三卡连线/画廊网格 | 新组件 `MediaGrid`/`MediaTriptych` | 数据驱动多 MediaFrame 组合版式，先吃自绘占位，P0-B 网页截图管线落地后即插真图 |
| 4 | **"人"当主画面用** | 用法规则 | 竞品全屏出镜占比远高于我们；CHAPTER-CRAFT 提高 host 时刻建议频率（每 2-3 章一次），新增两侧浮动图标道具卡 `IconProp` 小组件 |
| 5 | 局部 UI 放大浮层 | MediaFrame 变体 | `MediaFrame` 加 `callout` 子区域放大模式（截图上叠局部放大卡）——依赖 P0-B 真截图，随素材管线一起做 |
| 6 | AI 伪实拍 B-roll / 剧情段 | P2 图/视频生成 | 成本与风格一致性待验证，维持 P2 后置（与既有裁决一致） |

**明确不学**：花字字幕（竞品自己都不用）、竖屏改版（该账号同为横屏 16:9 内容+上下黑边直发抖音，与我们现状一致——此前"竖屏是最大差距"的判断按此实证下调）。

## 四、建议落地顺序

FlowDiagram 发光图解（纯组件，命中最高频套路）→ BGM+ducking（频谱实证的差距，机制与 SFX 同构）→ MediaGrid 组合版式（先占位后真图）→ host 频率+IconProp（规则层零成本）→ P0-B 网页截图管线（素材真源）。

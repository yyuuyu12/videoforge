### 发现与修复

1. outline 新格式不能预先统计章节，进度总数随目录增长。服务端现兼容旧时间表、阿拉伯数字标题及“开场/第X章”标题。
2. options API 接受 UI 不支持的主题并延迟到 scaffold 失败。现增加四个产品主题白名单并返回明确 400。
3. 单行 `narrations=[...]` 被工作台误报 0 step。现统计数组内字符串字面量，#20 正确显示 1/4/4/4/4/5。
4. `extract-narrations.ts` 不接受压缩后的 `from"..."` 导入。方法论源库、产品快照和 #20 当前工程同步修复，重试后提取 22 段。
5. 数字人布局审计漏识别 `right:448px` 的绝对定位预留。现视为有效右侧预留，#20 6/6 章通过。
